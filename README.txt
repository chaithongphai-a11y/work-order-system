ระบบใบงาน

รูปแบบเลขที่บิล
- คนที่ 1 ใช้ตัวอักษร A
- คนที่ 2 ใช้ตัวอักษร B
- คนที่ 3 ใช้ตัวอักษร C
- คนที่ 4 ใช้ตัวอักษร D
- ระบบจะต่อด้วยเดือนและเลขรัน 3 หลัก เช่น B07/001
- ถ้าเลือกคนลงข้อมูลแล้วกด "ออกเลข" ระบบจะหาเลขถัดไปให้อัตโนมัติ

วิธีเปิดใช้งาน
1. เปิด PowerShell ในโฟลเดอร์นี้
2. รันคำสั่ง:
   C:\Users\chait\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe server.js
3. เปิดเว็บ:
   http://localhost:8787

ให้คนอื่นในวง Wi-Fi เดียวกันเข้าใช้งาน
1. เปิดระบบไว้บนเครื่องหลัก
2. ดู Network URL ที่แสดงในหน้าต่าง PowerShell เช่น http://192.168.1.20:8787
3. ให้เครื่องอื่นเปิด URL นั้นในเบราว์เซอร์

ข้อมูลใบงานจะเก็บอยู่ที่:
data/jobs.json

ควรสำรองไฟล์ data/jobs.json เป็นระยะ ถ้าใช้เป็นข้อมูลจริงประจำวัน
