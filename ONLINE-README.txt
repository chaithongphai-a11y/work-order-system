คู่มือทำเป็นระบบออนไลน์

ระบบนี้พร้อมนำไปอัปขึ้นโฮสต์ที่รองรับ Node.js ได้แล้ว

ไฟล์สำคัญ
- server.js
- package.json
- public/
- data/jobs.json

บัญชีเริ่มต้น
- admin / admin123
- person1 / 1111  ใช้เลขบิลขึ้นต้น A
- person2 / 2222  ใช้เลขบิลขึ้นต้น B
- person3 / 3333  ใช้เลขบิลขึ้นต้น C
- person4 / 4444  ใช้เลขบิลขึ้นต้น D
- account / account123  บัญชีลงยอดโอน
- sales / sales123  เซลล์ดูงาน
- manager / manager123  หัวหน้าแผนกตรวจสอบงาน

การจัดการผู้ใช้
- เข้าด้วย admin แล้วกด "จัดการผู้ใช้"
- เพิ่มบัญชีใหม่ได้เอง
- เลือกสิทธิ์ได้: ผู้ลงใบงาน, บัญชีลงยอดโอน, เซลล์ดูงาน, หัวหน้าแผนก, ผู้ดูแลระบบ
- บัญชีลงยอดโอนแก้ได้เฉพาะสถานะโอน
- เซลล์ดูงานดูรายการได้อย่างเดียว
- หัวหน้าแผนกดูและตรวจแก้งานได้
- ผู้ดูแลระบบจัดการผู้ใช้และลบใบงานได้

ควรเปลี่ยนรหัสผ่านก่อนใช้งานจริง
ตอนนี้ระบบเก็บบัญชีในไฟล์:
data/users.json

ให้เปลี่ยนรหัสผ่านผ่านหน้า "จัดการผู้ใช้" หรือแก้ไฟล์ data/users.json โดยตรงก่อนเอาขึ้นออนไลน์จริง

วิธีรันบนเครื่อง
1. เปิด PowerShell ในโฟลเดอร์นี้
2. รัน:
   C:\Users\chait\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe server.js
3. เปิด:
   http://localhost:8787

วิธีอัปขึ้นออนไลน์แบบง่าย
1. สมัครโฮสต์ที่รัน Node.js ได้ เช่น Render, Railway, Fly.io, VPS หรือเครื่องเซิร์ฟเวอร์ของร้าน
2. อัปโหลดทั้งโฟลเดอร์ work-order-system
3. ตั้ง Start Command เป็น:
   npm start
4. ตั้ง Environment Variables รหัสผ่านตามรายการด้านบน
5. ถ้าใช้โฮสต์แบบข้อมูลหายเมื่อรีสตาร์ต ต้องเปิด Persistent Disk และให้โฟลเดอร์ data/ อยู่ในดิสก์ถาวร

ข้อมูลจริงเก็บอยู่ใน:
data/jobs.json

ถ้าใช้งานจริงทุกวัน ควรสำรองไฟล์ data/jobs.json เป็นประจำ
